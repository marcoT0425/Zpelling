# Zpelling
A multi-spelling generator for words which have more than one accepted spelling.

The data is searched with WordNet precisely. The code is probably made by AI in order to generate a wordlist for words with more than one spelling.

There are some tags, include:

- `[R]`: Regional variant (eg colour, centre)
- `[S]`: Spelling variant (eg yogurt, adviser)
- `[A]`: Accent (Diacritic) variant (eg café, naive)
- `[H]`: Hyphen variant (eg pre-empt, coordinate)
- `[C]`: Compounding (eg ice cream, website)
- `[CC]`: Capitalisation (eg internet, lied)
- `[P]`: Pluralisation (eg formulae, statiums)
